# AIGRC Specification Family

Standards and protocols that define the AIGRC platform.

## Specifications

| ID | Title | Status | Description |
|----|-------|--------|-------------|
| [SPEC-CGA-001](./SPEC-CGA-001-CERTIFIED-GOVERNED-AGENT.md) | Certified Governed Agent Framework | Draft | CGA certification levels, golden thread protocol, governance lock |
| [AIGRC-EVT-001](./AIGRC-EVT-001-GOVERNANCE-EVENT-ARCHITECTURE.md) | Governance Event Architecture | Draft v0.2.0 | Event schema, deterministic identity, delivery channels, integrity |
| [AIGRC-API-001](./AIGRC-API-001-GOVERNANCE-API-CONTRACT.md) | AIGOS Governance API Contract | Draft v0.1.0 | HTTP API contract for event ingestion, retrieval, compliance, and integrity |

## Reading Order

1. **SPEC-CGA-001** — Understand the governance model (asset cards, golden thread, conformance levels)
2. **AIGRC-EVT-001** — How governance events flow between tools and the control plane
3. **AIGRC-API-001** — The HTTP API contract that implements the event architecture
